# Barbarian App

Foobar is a Python library for dealing with word pluralization.

## Installation

Instal NPM dulu

```bash
npm install 
```

Buka dua windows di Vscode (powershell)

### Buat run webnya
```bash
npm run dev
```

### Buat run Tailwind(stylingnya)
```bash
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```
